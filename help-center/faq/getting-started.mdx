## Steps to Start

1. Sign in at `https://deniirizbaev.com`
2. Click "New Project"
3. Enter a project name
4. Add team members if needed
5. Start creating pages in the sidebar
6. Use Markdown or rich editor for content

## Tips

Save often, and share the link with your collaborators. All changes are instant and visible!
