## Overview

Welcome to Deni Irizbaev's documentation platform. You get a centralized space to organize, manage, and share all your project documentation effortlessly. Build knowledge bases, API references, and guides tailored to your needs with intuitive tools and seamless collaboration.

This platform empowers you to create structured docs, embed code examples, and maintain version history without complexity. Start documenting your projects today and keep your team aligned.

## Quick Start

Get up and running in minutes with these essential steps.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://deni-irizbaev.example.com/register` with your email.
  </Step>
  <Step title="New Workspace" icon="plus">
    Click **New Workspace** and name it for your project.
  </Step>
  <Step title="Add First Page" icon="file-plus">
    Select **New Page**, choose `introduction.mdx`, and start editing.
  </Step>
  <Step title="Publish" icon="globe">
    Hit **Publish** to make your docs live and share the URL.
  </Step>
</Steps>

<Callout kind="tip">
  Pro tip: Use the preview mode to test components like `<Columns>` before publishing.
</Callout>

## Brand Guidelines

Maintain consistency with your branding. The primary brand color is `#3B82F6`.

<CodeGroup tabs="CSS,JSX">
  ```css
  :root {
    --brand-primary: #3B82F6;
  }
  
  .button {
    background-color: var(--brand-primary);
    color: white;
  }
  ```
  ```jsx
  const theme = {
    primary: '#3B82F6',
  };
  
  <Button style={{ backgroundColor: theme.primary }}>
    Click Me
  </Button>
  ```
</CodeGroup>

<Expandable title="Advanced Customization" default-open="false">
  Override defaults in your workspace settings:
  
  | Property       | Value      | Usage                  |
  |----------------|------------|------------------------|
  | Primary Color  | `#3B82F6` | Buttons, links, accents |
  | Secondary Color| `#1E40AF` | Hover states           |
  | Logo URL       | Custom     | Header branding        |
  
  Update via **Settings > Branding**.
</Expandable>
